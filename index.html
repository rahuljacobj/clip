TELEGRAM_SETUP
Step 1 — Create a bot (30 seconds)
Open Telegram → search @BotFather → send /newbot
Give it any name → copy the bot token it gives you

Step 2 — Get your Chat ID
Send any message to your new bot
Open this URL in your browser (replace TOKEN):
https://api.telegram.org/botTOKEN/getUpdates
Find "chat":{"id":XXXXXXXXX} — that number is your Chat ID

Step 3 — Add to Cloudflare + set webhook
Go to Cloudflare → your Worker → Settings → Variables → add:
TELEGRAM_BOT_TOKEN = your bot token
TELEGRAM_CHAT_ID = your chat ID number
Click Save and Deploy

Then open this URL once to register the webhook (replace TOKEN):
https://api.telegram.org/botTOKEN/setWebhook?url=https://gentle-sea-9a71.iamrahuljacobrj.workers.dev/telegram-webhook

Done! Send anything to your bot → appears here instantly.
